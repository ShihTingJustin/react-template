# react-template
A React.js template built without CRA

## Setup
1. clone repository
```
git clone https://github.com/ShihTingJustin/react-template.git
```
2. go to project directory
```
cd react-template
```
3. install dependencies
```
yarn
```
4. serving the project
```
yarn dev
```
5. use it on
```
http://localhost:3000/
```

## Stack
* Webpack
* Babel
* TypeScript
* React
* Tailwind
* Jest
* React Testing Library
